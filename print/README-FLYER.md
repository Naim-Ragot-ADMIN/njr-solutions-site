# Flyer NJR Solutions (Impression)

## Fichiers créés

- `print/flyer-a5.html` : flyer **A5** prêt impression.
- `print/flyer-a4-decoupable.html` : planche **A4 paysage** avec **2 flyers A5** à découper.
- `print/flyer-a4-x4-a6.html` : planche **A4 portrait** avec **4 flyers A6** à découper.
- `print/flyer.css` : styles print premium (noir/blanc avec touches dorées/bleu).
- `public/print/qr-njrsolutions.png` : QR code vers `https://njrsolutions.fr`.

## Hiérarchie visuelle appliquée

1. **Accroche forte** en haut pour capter l'attention.
2. **Promesse claire** (particuliers + professionnels, techno simplifiée).
3. **Services** en grille courte et lisible.
4. **Problèmes / solutions** concrets pour favoriser l'identification.
5. **Réassurance** en badges rapides.
6. **Bloc contact ultra visible** (téléphone dominant + site + QR).
7. **Zone d'intervention** clairement affichée.

## Impression recommandée

1. Ouvrir le fichier HTML voulu dans un navigateur.
2. `Ctrl + P`
3. Mettre :
   - **Taille réelle / 100%**
   - **Arrière-plans activés**
   - Marges : **aucune** (ou minimales selon imprimante)
4. Pour la version A4 découpable, découper sur le trait central.
